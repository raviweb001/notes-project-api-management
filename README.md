# Notes Management App 🚀

A sleek, full-stack web application to securely create, manage, and filter your day-by-day logs. Built with a modern "Premium Dark Tech" UI and a Node.js/Express backend.

## ✨ Features
* **Create Logs:** Add daily notes with a custom title, date, and description.
* **Filter by Date:** Easily search and filter your past notes using the date picker.
* **Delete Notes:** Remove unwanted logs with a clean confirmation prompt.
* **Premium UI/UX:** Features a modern dark glassmorphism design, custom scrollbars, and smooth hover animations.
* **Toast Notifications:** Real-time success and error pop-ups for better user feedback.
* **Responsive Design:** Adapts beautifully to different screen sizes.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Glassmorphism, Flexbox), Vanilla JavaScript (ES6+), FontAwesome Icons.
* **Backend:** Node.js, Express.js.
* **Architecture:** RESTful API architecture handling GET, POST, and DELETE requests.
